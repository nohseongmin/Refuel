# Refuel — Google Play 출시 가이드 (계정: 신규 개인 계정 기준)

> ⚠️ 신규 개인 계정은 프로덕션 전 "비공개 테스트 12명×14일" 의무 (2024-12 정책).
> 타임라인: 오늘 업로드 → 14일 테스트 → 프로덕션 액세스 신청(~7일 심사) → 출시. 총 ~3주.

## 준비물 (전부 준비돼 있음)
| 항목 | 위치 |
|---|---|
| 앱 번들 | `android\Refuel-release.aab` |
| 스토어 문구 | `android\store-listing.md` |
| 아이콘 512 | `docs\icon-512.png` |
| 피처 그래픽 | `android\feature-graphic.png` |
| 개인정보처리방침 | https://nohseongmin.github.io/Refuel/privacy.html |
| 스크린샷 2장+ | 폰에서 캡처 (아래 5-4) |

## 1. 앱 만들기 — https://play.google.com/console
모든 앱 → **앱 만들기**: 이름 `Refuel — AI 토큰 연료 게이지` / 기본 언어 한국어 / **앱** / **무료** / 선언 체크 → 만들기

## 2. 대시보드 "앱 설정" 설문 (답 그대로)
| 설문 | 답 |
|---|---|
| 앱 액세스 권한 | **전체/일부 기능 제한됨** → 안내: "PC 컴패니언 앱. PC 프로그램(github.com/nohseongmin/Refuel 릴리스)과 QR 페어링 후 데이터 표시. 페어링 전 안내 화면 동작 확인 가능" |
| 광고 | 광고 없음 |
| 콘텐츠 등급(IARC) | 이메일 입력 → 카테고리 **유틸리티** → 전부 "아니요" → 전체이용가 |
| 타겟층 | **18세 이상** (아동 대상 아님 → 심사 단순) |
| 뉴스 앱 | 아니요 |
| 데이터 보안 | 아래 5절 참고 — 옵트인 폰 연동이 데이터를 기기 밖으로 전송하므로 **정직하게 선언** |
| 정부 앱 / 금융 / 건강 | 아니요/해당 없음 |

## 3. 스토어 등록정보 (성장 → 스토어 등록정보)
store-listing.md에서 앱 이름/짧은 설명/자세한 설명 복붙 → 아이콘·피처 그래픽 업로드 → 스크린샷 업로드 → 개인정보처리방침 URL 입력 → 저장

## 4. 비공개 테스트 + AAB 업로드 (테스트 및 출시 → 비공개 테스트)
> 내부 테스트가 아니라 **비공개(Closed) 테스트** — 12명×14일 요건은 비공개 트랙만 인정.

새 버전 만들기 → **Play 앱 서명 동의(중요)** → `Refuel-release.aab` 업로드 → 출시 노트 → 저장/검토 → 비공개 테스트로 출시

**업로드 직후**: 설정 → 앱 무결성 → **앱 서명 키 인증서 SHA-256** 복사 → 나(Claude)에게 전달 → assetlinks.json 추가(주소창 제거 마무리)

## 5. 테스터 12명 모집
- 트랙 → 테스터 탭 → 이메일 목록 생성(구글계정 이메일들) → 저장 → **참여 링크 복사** → 공유
- 테스터: 링크에서 수락 → Play에서 설치
- 후보: 파비스팀, 창업동아리, 유르타팀, 학과 동기 (본인+가족 포함 가능)
- **14일 연속 옵트인 유지** 필요. 중도 이탈 대비 14~15명 확보 권장
- 유료 테스터 알선 사이트는 사기·정지 위험 — 지인으로

## 6. 14일 후: 프로덕션 액세스 신청
대시보드에 "프로덕션 액세스 신청" 활성화 → 설문(테스트 과정·피드백·개선점 서술) → 제출 → 심사 ~7일
요건 상세: https://support.google.com/googleplay/android-developer/answer/14151465

## 7. 프로덕션 출시
테스트 및 출시 → 프로덕션 → 새 버전 → 같은 AAB(또는 개선판) → 국가 선택(전체 권장) → 출시(심사 1~7일)

## 문제 생기면
- 업로드 거부/오류 → 메시지 그대로 Claude에게
- 버전 올릴 땐 twa-manifest.json의 appVersionCode +1 후 재빌드 요청
- 공식 도움말: 앱 서명 https://support.google.com/googleplay/android-developer/answer/9842756 · 데이터 보안 https://support.google.com/googleplay/android-developer/answer/10787469 · 비공개 테스트 https://support.google.com/googleplay/android-developer/answer/9845334
