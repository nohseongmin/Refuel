Refuel — AI 토큰 연료 게이지

## 짧은 설명 (80자)

## 자세한 설명 (4000자)
Claude Code 같은 AI 코딩 도구를 쓰다 보면 이런 경험, 있으시죠?

- 한창 작업 중인데 토큰 한도에 걸려 뚝 끊김
- "리셋이 몇 시였지?" 기억 안 나서 괜히 들락날락
- 5시간은 풀렸는데 주간 한도에 걸려 있는 줄 몰랐음

Refuel은 AI 코딩 도구의 "연료 게이지"입니다.

⏳ 재충전 카운트다운 — 5시간 윈도우가 언제 풀리는지 실시간으로
📅 주간 한도 추적 — 주간이 병목이면 자동으로 주간 카운트다운 표시
🔮 소진 예측 — 지금 속도면 언제 한도에 닿는지 미리 경고
📊 사용량 대시보드 — 오늘/이번 주/최근 7일, 입력·출력·캐시 분리
🔔 알림 — 재충전 완료·리셋 임박·사용량 경고를 폰으로

## 데이터 보안 섹션 답변 가이드 (Play Console) — 정직 버전
> 옵트인 '폰 연동'이 데이터를 기기 밖(ntfy)으로 전송하므로 "수집 없음"이 아니라 정직하게 선언한다.
> (틀린 선언 = 정책 위반·앱 삭제. 정직 선언은 손해 0이고 여전히 프라이버시 최강으로 보임.)

- 데이터를 수집/공유하나요? → **예** (옵트인 폰 연동 시 기기 밖 전송)
- 데이터 유형: **앱 활동 → 기타 작업**(토큰 사용량 수치·시각). ※ 코드/프롬프트/개인정보 아님
- 수집 목적: **앱 기능**
- 필수/선택: **선택**(옵트인, 기본 꺼짐)
- 전송 중 암호화: **예**(AES-GCM 종단간)
- 제3자 공유: **아니오**(ntfy는 중계자, 암호문만 통과·개발자 접근 불가)
- 삭제 요청: 서버 영구저장 없음(중계 캐시는 자동 만료). 설정에서 토픽·키 재발급으로 즉시 무효화 가능
- 계정 삭제 URL: 해당 없음(계정 없음)

강력 근거: 서버(ntfy)에 예약·보관되는 유일한 평문 메시지는 수치 없는 "재충전 완료" 알림뿐.
사용량 % 등 수치는 (a) 상태채널서 E2E 암호화 (b) 알림은 PC 켜져 있을 때만 실시간 → 서버 미보관.
