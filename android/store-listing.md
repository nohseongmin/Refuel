# Play 스토어 등록 문구 (복붙용)

## 앱 이름 (30자)
Refuel — AI 토큰 연료 게이지

## 짧은 설명 (80자)
AI 코딩 도구의 토큰 사용량·재충전 시간을 한눈에. 리셋되면 알림으로 알려드려요.

## 자세한 설명 (4000자)
Claude Code 같은 AI 코딩 도구를 쓰다 보면 이런 경험, 있으시죠?

- 한창 작업 중인데 토큰 한도에 걸려 뚝 끊김
- "리셋이 몇 시였지?" 기억 안 나서 괜히 들락날락
- 5시간은 풀렸는데 주간 한도에 걸려 있는 줄 몰랐음

Refuel은 AI 코딩 도구의 "연료 게이지"입니다.

⏳ 재충전 카운트다운 — 5시간 윈도우가 언제 풀리는지 실시간으로
📅 주간 한도 추적 — 주간이 병목이면 자동으로 주간 카운트다운 표시
🔮 소진 예측 — 지금 속도면 언제 한도에 닿는지 미리 경고
📊 사용량 대시보드 — 오늘/이번 주/최근 7일, 입력·출력·캐시 분리
🔔 알림 — 재충전 완료·리셋 임박·사용량 경고를 폰으로

🔒 프라이버시 최우선
- 개인정보 수집 없음, 계정 없음, 광고 없음
- 코드·대화 내용은 절대 전송되지 않음 (토큰 수치만)
- 상태 데이터는 종단간 암호화(AES-GCM)
- 전체 오픈소스: github.com/nohseongmin/Refuel

📌 사용 방법
1. PC에 Refuel 데스크톱 앱 설치 (GitHub 릴리스에서 무료)
2. 이 앱에서 QR 스캔으로 페어링
3. 끝 — 알아서 감지하고 알아서 알려줍니다
