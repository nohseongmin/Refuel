Refuel 개인정보처리방침 (Privacy Policy)

시행일: 2026-07-10

요약

Refuel은 개인정보를 수집하지 않습니다. 계정도, 로그인도, 광고도, 분석 도구도 없습니다.

1. 수집하는 정보

  • 앱(모바일)은 사용자의 어떤 개인정보도 수집·저장·전송하지 않습니다.
  • PC 프로그램은 사용자 PC에 이미 저장된 AI 코딩 도구의 로컬 사용 기록에서 토큰 수·시각·도구 이름만 읽어 집계합니다. 코드·대화 내용·API 키는 읽지 않습니다.

2. 전송되는 데이터 (폰 연동을 켠 경우에만)

  • 전송 항목: 토큰 사용량 수치, 시각, 도구 이름 — 이것이 전부입니다.
  • 해당 데이터는 AES-GCM으로 종단간 암호화되어 오픈소스 릴레이(ntfy)를 경유합니다. 릴레이 서버는 암호문만 볼 수 있으며, 복호화 키는 사용자의 PC와 폰에만 존재합니다.
  • 폰 연동은 기본 꺼짐(옵트인)이며 언제든 끌 수 있습니다.

3. 제3자 제공·광고·추적

없습니다. 광고 SDK, 분석(Analytics) SDK, 추적 기술을 일절 사용하지 않습니다.

4. 데이터 보관·삭제

  • 모든 설정·기록은 사용자 기기(PC의 ~/.refuel, 폰 브라우저 저장소)에만 보관됩니다.
  • 앱 삭제 또는 브라우저 데이터 삭제로 완전히 제거됩니다. 서버에 남는 것이 없습니다.

5. 오픈소스

전체 소스코드가 공개되어 있어 위 내용을 직접 검증할 수 있습니다: github.com/nohseongmin/Refuel
