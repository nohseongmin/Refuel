Refuel 개인정보처리방침 (Privacy Policy)

시행일: 2026-07-10

요약

계정도 로그인도 없고, 개발자는 사용자의 코드·프롬프트·개인정보를 수집하지 않습니다. 다만 Android 앱에는 광고(Google AdMob)가 포함되어 있어, 광고 제공을 위해 Google이 광고 식별자를 사용합니다(3절 참고). 웹 대시보드에는 광고가 없습니다.

1. 수집하는 정보

  • 개발자는 사용자를 식별할 수 있는 개인정보(이름·이메일·계정 등)를 수집하지 않습니다. 광고 관련 정보는 Google AdMob이 처리하며 3절에서 설명합니다.
  • PC 프로그램은 사용자 PC에 이미 저장된 AI 코딩 도구의 로컬 사용 기록에서 토큰 수·시각·도구 이름만 읽어 집계합니다. 코드·대화 내용·API 키는 읽지 않습니다.

2. 전송되는 데이터 (폰 연동을 켠 경우에만)

  • 전송 항목: 토큰 사용량 수치, 시각, 도구 이름 — 이것이 전부입니다.
  • 해당 데이터는 AES-GCM으로 종단간 암호화되어 오픈소스 릴레이(ntfy)를 경유합니다. 릴레이 서버는 암호문만 볼 수 있으며, 복호화 키는 사용자의 PC와 폰에만 존재합니다.
  • 폰 연동은 기본 꺼짐(옵트인)이며 언제든 끌 수 있습니다.

3. 광고 (Android 앱)

  • Android 앱에는 Google AdMob 광고가 포함되어 있습니다. 이를 위해 Google이 광고 식별자(Advertising ID) 및 기기·이용 정보를 수집·처리할 수 있습니다.
  • 광고는 앱 하단 배너 형태이며, 광고 노출을 위해 필요한 최소 정보만 사용됩니다. 개발자는 광고 목적의 사용자 데이터를 직접 수집하거나 보관하지 않습니다.
  • Google의 데이터 처리 방식: Google 파트너 사이트 정책
  • 개인 맞춤 광고는 기기 설정(Android → 설정 → Google → 광고)에서 재설정하거나 선택 해제할 수 있습니다.
  • 웹 대시보드(PWA)에는 광고가 없습니다. 광고는 Android 앱에만 포함됩니다.

3-1. 분석·추적

분석(Analytics) SDK나 사용자 행동 추적 기술은 사용하지 않습니다. 코드·프롬프트·대화 내용은 광고를 포함한 어떤 목적으로도 수집·전송되지 않습니다.

4. 데이터 보관·삭제

  • 모든 설정·기록은 사용자 기기(PC의 ~/.refuel, 폰 브라우저 저장소)에만 보관됩니다.
  • 앱 삭제 또는 브라우저 데이터 삭제로 완전히 제거됩니다. 서버에 남는 것이 없습니다.
