# Refuel

AI 코딩 에이전트의 토큰 사용량과 "재충전(5시간 윈도우)" 시간을 알려주는 데스크톱 모니터.
서버·계정 연결 없이 **로컬 로그만** 읽어 동작 — 코드/프롬프트는 PC 밖으로 나가지 않는다.

현재: Claude Code 지원. Codex는 실험적(`~/.codex` 있으면 자동). Cursor·Copilot은 사용량이
서버에만 있어 로컬 파싱 불가(추후 API 단계).

## 기능
- 에이전트별 카드 — 접으면 한 줄(리셋 카운트다운), 클릭하면 펼쳐져 상세
- 재충전까지 남은 시간(5h 윈도우) + 리셋 시각, 리셋 가까운 순 정렬
- 현재 윈도우 **입력/출력/캐시 분리**(캐시 때문에 부풀려진 숫자 구분), 오늘·이번 주, 최근 7일
- 한도 자동 추정(과거 윈도우 최댓값 기준) → %게이지·사용량 경고
- 트레이 상주: 닫으면 트레이로, 호버 툴팁에 카운트다운, 우클릭 종료로만 완전 종료
- 토스트 알림: 재충전 완료 · 리셋 임박 · 사용량 경고 (에이전트별)
- 단일 인스턴스(중복 실행 방지), 자동 실행(트레이로 조용히), 고해상도 선명(DPI)

## 개발 실행
```bash
pip install -r requirements.txt
python run.py
```

## exe 빌드 / 배포
```bat
build.bat
```
결과물 `dist\Refuel.exe`. 정식 배포는 태그 push로 자동(GitHub Actions):
```bash
git tag v0.5.0 && git push origin v0.5.0
```
→ [Releases](https://github.com/nohseongmin/Refuel/releases) 에 exe 자동 첨부.

> 미서명 exe라 처음 실행 시 Windows SmartScreen 경고가 뜰 수 있다 →
> "추가 정보 → 실행"으로 진행. (코드서명 인증서는 유료라 추후)

## 데이터 위치 (전부 로컬)
- 설정: `~/.refuel/config.json`
- 히스토리: `~/.refuel/history.db` (일별 토큰만, 로그 삭제돼도 보존)
- 로그: `~/.refuel/refuel.log`

## 설정 (앱 내 ⚙)
주간 리셋 요일/시각 · 닫으면 트레이로 · 자동 실행 · 강조 색상 · 테스트 알림.
